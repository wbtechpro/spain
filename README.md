# Карта Испании

Интерактивная карта с реальными открытыми данными: точки (школы, больницы,
йога, конный спорт, центры испанского) из OpenStreetMap и тепловые карты
(интернет, аренда, hate crimes) из официальных источников.

## Как запустить

```bash
# 1. Установить Python-зависимости (DuckDB + pdfplumber)
uv sync

# 2. Сгенерировать данные (можно параллельно)
uv run python scripts/fetch_internet_speed.py   # ~2-3 мин (скачивает Ookla parquet ~600 МБ)
uv run python scripts/fetch_rent.py             # ~5 сек
uv run python scripts/fetch_hate_crimes.py      # ~10 сек

# 3. Локальный сервер
python3 -m http.server 5173       # или `npx serve -l 5173 .`
open http://localhost:5173
```

## Структура

```
index.html                  — сама карта (Leaflet + плагины)
data/
  internet_speed.json       — Ookla Q4 2025, 5333 ячейки по 10 км
  rent_cost.json            — SERPAVI 2023, 50 провинций
  hate_crimes.json          — МВД Испании 2024, 52 провинции
  raw/                      — исходные PDF/parquet (в git не коммитить)
scripts/
  fetch_internet_speed.py   — Ookla Open Data → агрегация по сетке 0.1°
  fetch_rent.py             — SERPAVI 2025 PDF → провинциальные медианы
  fetch_hate_crimes.py      — МВД PDF → провинциальные счётчики
  spain_provinces.py        — справочник 52 провинций + координаты
pyproject.toml              — Python-зависимости (uv)
.claude/launch.json         — конфиг preview-сервера
```

## Источники данных

| Слой                            | Источник                                        | Период  | Гранул.    | Формат |
|---------------------------------|-------------------------------------------------|---------|------------|--------|
| Скорость интернета              | Ookla Open Data (S3, CC BY-NC-SA 4.0)           | 2025-Q4 | 10-км тайл | Parquet |
| Стоимость аренды                | SERPAVI 2025 (Mineco/MIVAU)                     | 2023    | провинция  | PDF    |
| Преступления по ненависти       | Ministerio del Interior, informe 2024           | 2024    | провинция  | PDF    |
| Школы / больницы / йога / …     | OpenStreetMap (Overpass API)                    | live    | точки      | JSON   |
| Аэропорты / горные зоны / ски   | Курированный список в `index.html`              | —       | точки      | —      |

## Что пока не подключено

Слои помечены в панели зачёркнутым шрифтом «нет данных»:

- Перебои с электричеством — Red Eléctrica / CNMC публикуют SAIDI/SAIFI
  только по дистрибьюторам, не по координатам.
- Средняя цена часа йоги — нет открытого источника, потребует скрейпинга.
- Срок ожидания cita previa — не публикуется официально.
- Русскоязычные логопеды, нейрореабилитационные центры — нет реестра;
  ручная курация.

## OSM-слои — особенности

Overpass API медленно отвечает на большие bbox. Плотные категории (школы,
больницы) подгружаются только при zoom ≥ 7 для текущей видимой области.
При малом зуме статус покажет `zoom ≥ 7`.

Кнопка «↻ Обновить OSM для текущей области» перезапрашивает включённые слои
в новых границах карты.
